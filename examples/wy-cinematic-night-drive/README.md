# WY Cinematic Night Drive

This example is a self-contained cinematic single-page prototype. It uses scroll
position as a camera rail for skyline parallax, road motion, a procedural Canvas
star field, chapter cards, and responsive navigation controls.

## Files

- `index.html` - complete static source with inline CSS and JavaScript.
- `THIRD_PARTY_LICENSES.md` - asset and dependency notes.

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static file
server.

```bash
npx serve .
```

## Dependencies

None. The page does not import external scripts, images, fonts, or 3D models.
